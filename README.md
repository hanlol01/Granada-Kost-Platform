# Granada Kost Platform

Monorepo untuk platform Granada Kost.

- `apps/admin`: aplikasi admin untuk `admin.kostsaya.com`.
- `apps/penghuni`: aplikasi Penghuni mobile-first/PWA untuk `penghuni.kostsaya.com`.
- `backend/api`: rencana backend NestJS.
- `packages/ui`: komponen UI bersama.
- `packages/domain`: model domain, enum, schema, dan aturan bisnis bersama.
- `packages/api-client`: client API typed untuk frontend.
- `packages/config`: konfigurasi lint/build/env bersama.
- `docs`: dokumentasi produk, arsitektur, security, dan workflow.

Package manager standar: npm.
